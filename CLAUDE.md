# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

A browser-based animated cat widget with two deliverables: a simple blink-only version (`cat-player.html`) and a full interactive version (`cat.html`) that adds a click-triggered closeup animation, a meow sound, and a rotating quote system using the Tulados serif font.

## Key Files

- **`cat.html`** — The main deliverable. On load, the sprite blinks. Clicking the cat hides the sprite, plays a meow, draws a full-screen close-up (alternating between `closeframe1.png` and `closeframe2.png` on a canvas), and shows a shuffled quote above the cat. Clicking anywhere after that dismisses and resumes blinking.
- **`remove_bg.py`** — One-shot pipeline: reads `cat_video.mp4`, strips background with `rembg` (model `isnet-general-use`), writes RGBA frames to a temp dir, then assembles them into `cat_background_removed.webm` (VP9 + alpha) via the bundled ffmpeg from `imageio-ffmpeg`.
- **`tulados.ttf`** — Custom serif font loaded via `@font-face`, used for the quote overlay in `cat.html`.

## Required Assets (not generated by scripts)

These files must be present alongside the HTML for it to work:

| File | Purpose |
|---|---|
| `cat_blink.png` | Horizontal sprite strip for blink animation |
| `closeframe1.png` | First frame of close-up animation |
| `closeframe2.png` | Second frame of close-up animation |
| `soundreality-cat-meow-fx-461188.mp3` | Meow sound on click |
| `tulados.ttf` | Quote font |

## Sprite Sheet Convention

`cat_blink.png` is a horizontal strip of **6 frames**, each **185 × 202 px** (total width 1110 px). Frame 0 = eyes open; frames 1–3 = progressive eye-close. The JS snaps back to frame 0 immediately after frame 3 (no symmetric re-open) to avoid a double-blink artifact.

## Blink & Click Logic (`cat.html`)

- **Blink loop**: probabilistic; 25% chance of a slow blink (180 ms/step vs. 110 ms/step). Next blink fires 2.5–6 s after the previous one ends.
- **Click animation**: sets `clicking = true`, clears the blink timer, hides the `#cat` div, shows a `<canvas>` with `closeframe1.png`, plays the meow, swaps to `closeframe2.png` after 600 ms, then reverts to `closeframe1.png` when audio ends.
- **Quote system**: quotes are shuffled into a queue and consumed one at a time; the queue refills and reshuffles when exhausted.
- **Dismiss**: a single click anywhere (registered after a `setTimeout(0)` guard) hides the canvas, restores the sprite, and restarts the blink loop.

## Viewing

Open `cat.html` (or `cat-player.html`) directly in a browser — no server needed. All assets must be in the same directory.

## Running the Background Removal Script

```powershell
pip install opencv-python rembg pillow imageio-ffmpeg
python remove_bg.py
```

Requires `cat_video.mp4` in the working directory. Output is `cat_background_removed.webm`.

## Frame Preview Directories

`frames_preview/` and `frames_preview2/` contain intermediate per-frame PNGs generated during sprite sheet construction. They are scratch/reference artifacts, not loaded by the HTML.
